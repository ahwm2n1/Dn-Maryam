# Dietitian Maryam - Clinical Nutritionist Website

![Website Banner](https://via.placeholder.com/1200x400/c17b4c/ffffff?text=Dietitian+Maryam)

A complete professional website for **Dietitian Maryam Shahrukh**, a certified clinical nutritionist. This project includes a fully responsive frontend, admin panel, and MySQL-based backend with REST APIs.

## ✨ Features

### 🌐 **Frontend (3 Pages)**
- **Home Page** - Profile, certificates, blogs, FAQ, newsletter
- **Services Page** - All services with pricing, reviews, happy clients
- **Appointment Page** - Booking form with WhatsApp integration

### ⚙️ **Backend API**
- Node.js + Express + MySQL
- RESTful APIs for all data
- File upload support (profile images, blog images)
- JWT authentication for admin

### 👩‍💼 **Admin Panel**
- Secure login system
- Dashboard with stats
- Manage appointments (view, update status, delete)
- Manage services (add/edit/delete)
- Manage blog posts
- Update profile information
- View subscribers and visitors

### 📱 **Special Features**
- **WhatsApp Integration** - Appointment notifications
- **AI Chatbot** - Basic nutrition assistant
- **Newsletter Subscription**
- **Visitor Tracking**
- **Fully Responsive Design**
